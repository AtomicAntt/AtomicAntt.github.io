Cure the Shadow

Created with Godot

This game was created as a submission to Pirate Software Jam 15, featuring a stealth 2D platformer where you are cursed to hide in the shadows and must steal the cure. The theme of the game jam was: "Shadows and Alchemy".

I made this game as a solo developer using the Godot game engine with GDScript.

In total, there were 2,366 teams competing in the game jam.

Game Technical Features:

  • Developed radial raycasting to detect if players are within light sources.
  • Developed the player controller using a state machine.
  • Developed a skippable dialogue system that can cause events.
  • Created enemy AI behavior.
