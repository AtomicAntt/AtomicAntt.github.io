This game was created as a submission to Boss Rush Jam 2025, featuring a boss rush game where you collect and whirl your weapons to defeat each boss. The theme of the game jam was: "Spin".

I was a solo developer for this submission, using the Godot Engine.

Additionally, this game won 12th place for fun and 30th place for both boss design and controls. In total, 866 game jam teams participated in this game jam.

Game Technical Features:

  • Rotating item system for the player's main mechanic, which can dynamically add any # of items/shields.
  • Modular state machines for each boss.
  • Modular boss drops for each boss.
  • Screen shake system, with variable amount of screen shake.
