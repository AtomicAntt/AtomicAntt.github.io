This game was created as a submission to Goedware Jam #10, featuring a 2D platformer where you shoot a large laser to destroy your enemies.. and the level? The theme of the game jam was: "Consequences".

I contributed as the gameplay programmer, collaborating with an artist, using the Godot game engine with .NET (C#).

Additionally, this game won 2nd place (Jury rating) at Goedware Jam #10, awarding our team a total of $125!

Game Technical Features:

  • 6 different enemy AI behaviors and attacks using state machines.
  • Laser mechanic which calculates recoil and uses raycasting to destroy/convert tiles depending on type.
  • Skippable dialogue system.
  • Dedicated game scene switcher.
